---
title: Ordered List Tag - <ol>
description: This lesson discusses the Ordered List <ol> tag and its usage in web development.
---

This tag is used to create an ordered list in the
document. It contains a list of items that are displayed
with numbers or letters.

```html showLineNumbers
<ol type="A" start="3">
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>
```

The code above shows an example of the `<ol>` tag. This tag
is used to create an ordered list in the document, which contains a list
of items displayed with numbers or letters. The `<li>` tag is used
to define each item in the list. The `type` attribute is used to
specify the type of numbering to use, such as `1`, `A`,
`a`, `I`, or `i`. The default type is `1`.
The `start` attribute is used to specify the starting number of
the list. The default start number is `1`.
