---
title: Label Tag - <label>
description: This lesson discusses the Label <label> tag and its usage in web development.
---

This tag is used to create a label for an input field
in a form. It is used to provide a description or name
for the input field. Clicking on the label focuses the
associated input field.

```html showLineNumbers
<label for="name">Name:</label>
<input type="text" id="name" name="name" />
```

The code above shows an example of the `<label>` tag.
This tag is used to create a label for an input field in a form, which
provides a description or name for the input field. The `for`
attribute specifies the ID of the input field that the label is associated
with.
