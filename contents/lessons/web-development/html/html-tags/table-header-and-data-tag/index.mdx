---
title: Table Head and Data Tag - <th> and <td>
description: This lesson discusses the Table Head <th> and Table Data <td> tag and its usage in web development.
---

These tags are used to define header cells and data cells
in a table, respectively. The `th>` tag is used to
define a header cell, while the `td>` tag is used to
define a data cell.

```html showLineNumbers
<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Jane</td>
        <td>30</td>
    </tr>
</table>
```

The code above shows an example of the `<th>` and
`<td>` tags. The `<th>` tag is used to define a header cell
in a table, while the `<td>` tag is used to define a data cell in
a table.
