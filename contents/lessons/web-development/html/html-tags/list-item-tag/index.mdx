---
title: List Item Tag - <li>
description: This lesson discusses the List Item <li> tag and its usage in web development.
---

This tag is used to define an item in a list. It is
used inside the `<ul>` or `<ol>` tag to
define each item in the list.

```html showLineNumbers
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>
```

The code above shows an example of the `<li>` tag. This tag
is used to define an item in a list, which is displayed as part of an
unordered or ordered list.
