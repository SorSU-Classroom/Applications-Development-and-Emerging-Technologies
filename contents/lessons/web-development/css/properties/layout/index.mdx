## Layout

### Display

The `display` property is used to set the display behavior of an element.
The display property can be set to `block`, `inline`, `inline-block`, `flex`,
`grid`, or `none`.

<div className="mdx-preview mdx-preview-display">
  <div style={{ display: 'block' }}>
    <div style={{ width: '100px', height: '100px', backgroundColor: 'lightblue' }}>Item 1</div>
    <div style={{ width: '100px', height: '100px', backgroundColor: 'lightblue' }}>Item 2</div>
    <div style={{ width: '100px', height: '100px', backgroundColor: 'lightblue' }}>Item 3</div>
  </div>

  <div style={{ display: 'inline' }}>
    <div style={{ width: '100px', height: '100px', backgroundColor: 'lightblue' }}>Item 1</div>
    <div style={{ width: '100px', height: '100px', backgroundColor: 'lightblue' }}>Item 2</div>
    <div style={{ width: '100px', height: '100px', backgroundColor: 'lightblue' }}>Item 3</div>
  </div>

  <div style={{ display: 'flex' }}>
    <div style={{ width: '100px', height: '100px', backgroundColor: 'lightblue' }}>Item 1</div>
    <div style={{ width: '100px', height: '100px', backgroundColor: 'lightblue' }}>Item 2</div>
    <div style={{ width: '100px', height: '100px', backgroundColor: 'lightblue' }}>Item 3</div>
  </div>
</div>

### Flexbox

The `flex` property is used to set the flex behavior of an element. The
flex property can be set to `flex-grow`, `flex-shrink`, `flex-basis`, or
a combination of these values.

<div className="mdx-preview mdx-preview-flexbox">
  <div style={{ display: 'flex' }}>
    <div style={{ flex: 2, height: '100px', backgroundColor: 'lightblue' }}>Item 1</div>
    <div style={{ flex: 1, height: '100px', backgroundColor: 'lightcoral' }}>Item 2</div>
    <div style={{ flex: 1, height: '100px', backgroundColor: 'lightgreen' }}>Item 3</div>
  </div>
</div>

    ```html showLineNumbers
    <div class="container">
      <div class="item item1">Item 1</div>
      <div class="item item2">Item 2</div>
      <div class="item item3">Item 3</div>
    </div>
    ```
  </TabsContent>
  <TabsContent value="css">
    ```css showLineNumbers
    .container {
      display: flex;

    }

    .item2 {
      flex: 1;
    ```

The code above shows an example of the `flex` property. In this
example, the flex property of the `<div>` elements is set to 1, 2, and 1,
creating different flex behaviors for the elements.

### Grid

The `grid` property is used to set the grid behavior of an element. The
grid property can be set to `grid-template-columns`, `grid-template-rows`,
`grid-gap`, or a combination of these values.

<div className="mdx-preview mdx-preview-grid">
  <div className="container1" style={{ display: 'grid', gridTemplateColumns: '1fr 1fr 1fr', gridGap: '10px', marginBlock: "10px !important" }}>
    <div style={{ backgroundColor: 'lightblue' }}>Item 1</div>
    <div style={{ backgroundColor: 'lightcoral' }}>Item 2</div>
    <div style={{ backgroundColor: 'lightgreen' }}>Item 3</div>
  </div>

  <div className="container2" style={{ display: 'grid', gridTemplateColumns: '1fr 2fr 1fr', gridGap: '10px', marginBlock: "10px !important" }}>
    <div style={{ backgroundColor: 'lightblue' }}>Item 1</div>
    <div style={{ backgroundColor: 'lightcoral' }}>Item 2</div>
    <div style={{ backgroundColor: 'lightgreen' }}>Item 3</div>
  </div>

  <div className="container3" style={{ display: 'grid', gridTemplateColumns: '1fr 2fr 1fr', gridTemplateRows: '1fr 2fr 1fr', gridGap: '10px', marginBlock: "10px !important" }}>
    <div style={{ backgroundColor: 'lightblue' }}>Item 1</div>
    <div style={{ backgroundColor: 'lightcoral' }}>Item 2</div>
    <div style={{ backgroundColor: 'lightgreen' }}>Item 3</div>
    <div style={{ backgroundColor: 'lightblue' }}>Item 4</div>
    <div style={{ backgroundColor: 'lightcoral' }}>Item 5</div>
    <div style={{ backgroundColor: 'lightgreen' }}>Item 6</div>
    <div style={{ backgroundColor: 'lightblue' }}>Item 7</div>
    <div style={{ backgroundColor: 'lightcoral' }}>Item 8</div>
    <div style={{ backgroundColor: 'lightgreen' }}>Item 9</div>
  </div>
</div>
