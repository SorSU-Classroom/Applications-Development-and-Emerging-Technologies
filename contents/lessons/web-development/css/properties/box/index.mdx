## Box

### Margin

The `margin` property is used to set the margin of an element. The
margin is the space outside the border of the element and is used to
create space between elements.

<Tabs defaultValue="html" className="pt-5 pb-1">
  <TabsList>
    <TabsTrigger value="html">HTML</TabsTrigger>
    <TabsTrigger value="css">CSS</TabsTrigger>
  </TabsList>
  <TabsContent value="html">
    ```html showLineNumbers
    <div class="container">
      <div class="item">Item 1</div>
      <div class="item">Item 2</div>
      <div class="item">Item 3</div>
    </div>
    ```
  </TabsContent>
  <TabsContent value="css">
    ```css showLineNumbers
    .container {
      display: flex;
      flex-direction: column;
    }

    .item {
      margin: 10px;
      width: 100px;
      height: 100px;
      background-color: lightblue;
    }
    ```
  </TabsContent>
</Tabs>

The code above shows an example of the `margin` property. In this
example, the margin of the `<div>` element is set to 10 pixels, creating
space around the element.

### Space Between

The `space-between` property is used to set the space between elements
in a flex container. It distributes the extra space between elements in
the container.

<Tabs defaultValue="html" className="pt-5 pb-1">
  <TabsList>
    <TabsTrigger value="html">HTML</TabsTrigger>
    <TabsTrigger value="css">CSS</TabsTrigger>
  </TabsList>
  <TabsContent value="html">
    ```html showLineNumbers
    <div class="container">
      <div class="item">Item 1</div>
      <div class="item">Item 2</div>
      <div class="item">Item 3</div>
    </div>
    ```
  </TabsContent>
  <TabsContent value="css">
    ```css showLineNumbers
    .container {
      display: flex;
      justify-content: space-between;
    }

    .item {
      width: 100px;
      height: 100px;
      background-color: lightblue;
    }
    ```
  </TabsContent>
</Tabs>

The code above shows an example of the `space-between` property. In this
example, the `justify-content` property is set to `space-between` to
distribute the extra space between the elements in the `<div>` container.

### Width and Height

The `width` and `height` properties are used to set the width and
height of an element. The width and height can be set to a specific size,
such as pixels or percentages, or to `auto` to automatically adjust to
the content.

    <TabsTrigger value="css">CSS</TabsTrigger>

  <TabsContent value="html">
    ```html showLineNumbers
    <div class="container">
      <div class="item item1">Item 1</div>
      <div class="item item2">Item 2</div>
      <div class="item item3">Item 3</div>
    </div>
    ```
  </TabsContent>
  <TabsContent value="css">
    ```css showLineNumbers
    .container {
      display: flex;
      flex-direction: column;
    }

    .item {
      background-color: lightblue;
    }

    .item1 {
      width: 100px;
      height: 100px;

    .item2 {
      width: 50%;
      height: 50%;
    }

    .item3 {

      height: auto;
    ```

  </TabsContent>
</Tabs>

The code above shows an example of the `width` and `height` properties.
In this example, the width and height of the `<div>` elements are set to
100 pixels, 50%, and auto, creating different sizes for the elements.
