## Text

### Text Align

The `text-align` property is used to set the alignment of the text
within an element. The text can be aligned to the `left`, `right`, `center`,
or `justified`.

<div className="mdx-preview mdx-preview-text-align">
  <p style={{ textAlign: 'center' }}>This is a paragraph.</p>
</div>

<Tabs defaultValue="html" className="pt-5 pb-1">
  <TabsList>
    <TabsTrigger value="html">HTML</TabsTrigger>
    <TabsTrigger value="css">CSS</TabsTrigger>
  </TabsList>
  <TabsContent value="html">
    ```html showLineNumbers
    <p>This is a paragraph.</p>
    ```
  </TabsContent>
  <TabsContent value="css">
    ```css showLineNumbers
    p {
      text-align: center;
    }
    ```
  </TabsContent>
</Tabs>

The code above shows an example of the `text-align` property. In this
example, the text in the `<p>` element is aligned to the center.

### Text Decoration

The `text-decoration` property is used to set the decoration of the text
within an element. The text can be `none`, `underline`, `overline`,
`line-through`, or a combination of these values.

<div className="mdx-preview mdx-preview-text-decoration">
  <p style={{ textDecoration: 'underline' }}>This is a paragraph.</p>
</div>

<Tabs defaultValue="html" className="pt-5 pb-1">
  <TabsList>
    <TabsTrigger value="html">HTML</TabsTrigger>
    <TabsTrigger value="css">CSS</TabsTrigger>
  </TabsList>
  <TabsContent value="html">
    ```html showLineNumbers
    <p>This is a <span class="underline">paragraph</span>.</p>
    ```
  </TabsContent>
  <TabsContent value="css">
    ```css showLineNumbers
    .underline {
      text-decoration: underline;
    }
    ```
  </TabsContent>
</Tabs>

The code above shows an example of the `text-decoration` property. In this
example, the text in the `<span>` element with the class `underline` is
underlined.
