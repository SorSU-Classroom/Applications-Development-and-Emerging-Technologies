## Transforms

The `transform` property is used to apply transformations to elements
in CSS. Transformations can be used to rotate, scale, skew, or move
elements on a web page. The `transform` property can be used with
different values to apply different types of transformations.

### Scale

The `scale` value is used to scale an element. The scale value can be
set to a specific size, such as `1.5` or `0.5`, to scale the element up
or down.

<div className="mdx-preview mdx-preview-transforms">
  <div style={{
    display: 'flex',
    flexDirection: 'column',
  }}>
    <div style={{
      width: '100px',
      height: '100px',
      background: 'lightblue',
      transform: 'scale(1.5)'
    }}>Item 1</div>
    <div style={{
      width: '100px',
      height: '100px',
      background: 'lightcoral',
      transform: 'scale(0.5)'
    }}>Item 2</div>
    <div style={{
      width: '100px',
      height: '100px',
      background: 'lightgreen',
      transform: 'scale(2)'
    }}>Item 3</div>
  </div>
</div>

elements.

### Origin

The `transform-origin` property is used to set the origin of the
transformed element. The origin value can be set to a specific position,
such as `top left`, `center`, or `bottom right`, to change the origin of
the transformation.
