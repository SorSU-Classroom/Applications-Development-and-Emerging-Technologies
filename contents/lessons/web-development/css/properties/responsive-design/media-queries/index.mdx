## Media Queries

The `@media` rule is used to apply different styles for different media
types, such as screen sizes, devices, or orientations. Media queries can
be used to create responsive designs that adapt to different screen sizes.

<div className="mdx-preview mdx-preview-media-queries">
  <div className="container">
    <div className="item">Item 1</div>
    <div className="item">Item 2</div>
    <div className="item">Item 3</div>
  </div>
</div>

<Tabs defaultValue="html" className="pt-5 pb-1">
  <TabsList>
    <TabsTrigger value="html">HTML</TabsTrigger>
    <TabsTrigger value="css">CSS</TabsTrigger>
  </TabsList>
  <TabsContent value="html">
    ```html showLineNumbers
    <div class="container">
      <div class="item">Item 1</div>
      <div class="item">Item 2</div>
      <div class="item">Item 3</div>
    </div>
    ```
  </TabsContent>
  <TabsContent value="css">
    ```css showLineNumbers
    .container {
      display: flex;
      flex-direction: row;
      flex-wrap: wrap;
      gap: 12px;
    }

    .item {
      display: flex;
      justify-content: center;
      align-items: center;
      width: 150px;
      height: 150px;
      background-color: lightblue;
    }

    @media screen and (max-width: 600px) {
      .item {
        width: 100px;
        height: 100px;
      }
    }

    @media screen and (max-width: 400px) {
      .container {
        flex-direction: column;
      }
    }
    ```
  </TabsContent>
</Tabs>

The code above shows an example of media queries. In this example, the
flex direction of the `<div>` elements is changed to column when the screen width is less than
400 pixels, and the width and height of the elements are changed when the
screen width is less than 600 pixels.
