## Hover

The `hover` property is used to change the style of an element when the
mouse hovers over it. The hover property can be used to change the color,
background color, or other styles of the element.

<div className="mdx-preview mdx-preview-hover">
  <button>Hover Over Me</button>
</div>

<Tabs defaultValue="html" className="pt-5 pb-1">
  <TabsList>
    <TabsTrigger value="html">HTML</TabsTrigger>
    <TabsTrigger value="css">CSS</TabsTrigger>
  </TabsList>
  <TabsContent value="html">
    ```html showLineNumbers
    <button>Hover Over Me</button>
    ```
  </TabsContent>
  <TabsContent value="css">
    ```css showLineNumbers
    button {
      background-color: lightblue;
      color: white;
      padding: 10px 20px;
      border: none;
    }

    button:hover {
      background-color: lightcoral;
    }
    ```
  </TabsContent>
</Tabs>

The code above shows an example of the `hover` property. In this example,
the background color of the `<button>` element is changed to `lightcoral`
when the mouse hovers over it.
